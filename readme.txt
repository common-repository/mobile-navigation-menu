Make your Navigation Menu Responsively Awesome! Themes that have no supported mobile navigation menu or not fine looking mobile menu version, this your solution!

== Description ==
Mobile Navigation Menu is a sophisticated plugin to automatically create a mobile menu version for your existing themes for a very simple configuration. If you are not satisfied with the looks of your current mobile navigation menu or not even have a mobile menu version, this is your solution.

A simple and lightweight Responsive Web Design plugin that automatically pops your mobile navigation menu in a breakpoint set.

= Features and Usage =

Few of the many features:
<ul>
<li>Make Mobile Menu for Your Existing Theme</li>
<li>Slide Navigation Like an Iphone. See Screenshots</li>
<li>Set your Breakpoint. MNMenu pops!</li>
<li>Pick your Desired Background, Icon, and Text Color</li>
<li>You can upload your Company Logo</li>
<li>Font Awesome Installed. (You can disable this feature via admin)</li>
<li>Hide/Show Admin Menu</li>
<li>Can Customize Image Loader</li>
<li>..and More</li>
</ul>

More usage, demo, and documentation can be read on [DEMO](http://www.innovedesigns.com/wordpress/plugin/mobile-navigation-menu/ "DEMO")

== Installation ==

Installation is very easy, you can either upload or install it via WP plugin section (the easy way) or Upload using FTP to wp-content/plugins/

1. Of course, Activate it via plugin section! 
1. Go to your WP Dashboard > Tools > Mobile Navigation Menu (Just to see if it works :D)
1. The rest is just self-explanatory and You're done!
